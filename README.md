# auto-order



Скачайте проект

Відкрийте chrome://extensions/

Потрібно включити Режим розробника (Developer mode)

Нажми Завантажити розпаковане розширення (Load unpacked)

Вкажить папку з цим проектом

Відкрийте https://myigloo.is/
в розширеннях запустити Auto Form Filler

вкажить json текстове поле (скопіюйте в підказці)

натиснути "Запустити"
